Researcher / Practitioner / Connector

Ginzo Shizuki

世界の言説空間を、豊かに。

特定の現場（ローカル）に蓄えられた知を、まだそれを知らない社会へ。

— 銀蔵って、何してる人？

大学院では、組織や社会の現実が言葉と物語によってどう編まれるかを研究する。

現場では、複数の事業や活動を通じて、別のレンズを社会に差し込む。

狭まりゆく世界に、もう少しの風通しを。— 楽しく、しなやかに。

Selected work

複数の現場で、レンズを差し込む。

01 合同会社BWH総合研究所 Organization Design / Narrative / Dialogue →

組織を「人事最適化」ではなく、対話と物語として見るレンズ。人事・組織開発の実装支援。社会構成主義・物語論を背景に、対話を通じた組織変容に伴走します。

02 Kyobo合同会社 Board Game / Learning / Experience →

学校的な「正解のある、いやいやする学び」を、楽しく再構成する取り組み。京大ボドゲ製作所・京大マダミス製作所・Kyobo Kids。自社ボードゲーム開発と、法人向けゲーム制作を行います。

03 一般社団法人 e-donuts Inquiry / Children / Platform →

学びを「正解探し」ではなく、探究として開くレンズ。京都を拠点に、子どもの探究学習を支援。プログラム開発・教材開発・教育プラットフォーム構築に取り組みます。

04 株式会社（社名未定／BeautyAI） Beauty / Data / Communication •

リアル空間のコミュニケーションを、データとして可視化する実験。美容領域を起点に、対面コミュニケーションの構造化と新しい価値提示の方法を探る、収益事業としての実証実験。2026年6月登記予定。

About

Beliefs in Practice.

01

問題意識

効率、資本、学歴 — 単一の価値基準が、世界の見え方を狭めています。問題は「どの基準が正しいか」ではなく、レンズの選択肢が失われていることです。

02

ミッション

それぞれの現場（ローカル）に蓄えられた知識と、まだそれを知らない社会のあいだに立つ。別の見え方、別の現実の可能性を、楽しく提示し続けます。

03

立ち位置

思想と収益事業は両立します。資本主義を否定するのではなく、その内側で別のレンズを実装することを選びます。研究者であり、実践者であること。それが私のスタンスです。
